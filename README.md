# RNA-seq NGS Analysis Pipeline
This repository contains a complete single-end RNA-seq analysis pipeline.
## Tools Used
Cutadapt
HISAT2
Samtools
featureCounts
R (DESeq2)
## Workflow
1. Quality trimming using Cutadapt
2. Alignment using HISAT2
3. Gene counting
4. Differential expression analysis in R
## Dataset
NCBI SRA: SRR1552444
